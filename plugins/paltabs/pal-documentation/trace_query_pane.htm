The Query Tracing Controls Pane

The Tracing Controls Pane at the bottom of the Queries tab allows you to trace a built-in function or predicate during evaluation. The results of the tracing operation are displayed in the Protégé console window.

Note: The operation of this pane is identical to the Constraint Tracing Controls Pane.

 

 

This pane provides the following:

The Trace during evaluation checkbox allows you to enable and disable tracing.

The Function to Trace and Predicate to Trace menus allow you to choose the function or predicate you wish to trace. You can only trace one function and/or one predicate at a time.
