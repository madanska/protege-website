The Query Response Pane for Constraints

The Query Response pane at the right of the PAL Constraints Tab displays the list of frames that violated a selected constraint. Constraints which have violations are displayed with a red warning sign in the Status column after the Evaluate indicated constraints button in the Choose Constraints Pane has been clicked. 

Clicking on another constraint will show the frames that violate that constraint.

 

 

You can view the form for a frame by selecting the frame and then clicking the View button. 

If the Warn about constraints button in the Choose Constraints Pane is clicked, the Query Response pane is replaced by the Warnings and Errors Pane.
