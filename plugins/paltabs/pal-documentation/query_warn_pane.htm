The Query Warnings and Errors Pane

The Warnings and Errors pane at the right of the PAL Queries Tab displays syntax validation warnings and errors for a selected query. Queries which have warnings or errors are displayed with a red warning sign in the Status column after the Warn about indicated queries button has been clicked in the Choose Queries Pane.

This is an informational pane only. Clicking on another query will show the warnings for that query.

 

 

If the Evaluate indicated query button is pressed in the Choose Queries Pane, the Warnings and Errors pane is replaced by the Query Response Pane for Queries.
