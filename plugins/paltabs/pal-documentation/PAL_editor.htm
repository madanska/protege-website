The PAL Expression Editor

The PAL Expression Editor allows you to create PAL constraints in a semi-structured way.  Before you use the editor, you need to make sure that the PAL Expression Editor is set up on your constraints. See Configuring the PAL Expression Editor for more information.

The Expression Editor has four areas:

  • The Name entry bar, which allows you to type in a name for the constraint.
  • The Description entry box, which allows you to type in a description for the constraint.
  • The Statement area, which allows you to create and edit the statement part of a constraint.  
  • The Range area, which allows you to define a range for each variable in your constraint.

 

 

The Name and Description areas are simple text-entry boxes that can help you in naming and identifying your constraint. The Statement and Range areas are editors that give you access to structured information via such tools as right-mouse click menus (Command-click on the Macintosh), double-clicking on variables, and Editing buttons.
