Enabling PAL Queries

In order to use PAL queries, either via the PAL language or via the PAL Queries Tab, you must first include the pal-query project as part of your knowledge base. To do this:

  1. Copy the pal-query project files (pal-query.pprj, pal-query.pont, and pal-query.pins) from {your Protégé-2000 installation directory}/projects/pal-query folder to the folder or directory that contains the project in which you want to use PAL queries. 
  2. Go to Protégé and open your project. 
  3. Select Include from the Project menu.
  4. Include the pal-query project and click OK.

If you want to use the PAL Queries Tab to work with queries in an interactive way, you will need to enable the tab. See Enabling the PAL Constraints and PAL Queries Tabs for more information.
