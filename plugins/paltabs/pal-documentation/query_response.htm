The Query Responses Pane for Queries

The Query Response pane at the right of the The PAL Queries Tab displays the list of frames that satisfy a selected query. Queries which have queries that satisfy them are displayed with a set of plus signs after the Evaluate indicated queries button in the Choose Queries Pane has been clicked. 

Clicking on different query will show the the frames that satisfy that query.

 

 

You can view the form for a frame by selecting the frame and then clicking the View button. 

If the Warn about indicated queries button is pressed in the Choose Queries Pane, the Query Response pane is replaced by the Query Warnings and Errors Pane.
