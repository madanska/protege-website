Displaying Attached Constraints at a Class

Some constraints are attached to a specific class. These constraints are displayed in the Class Form for that class. To see attached constraints:

  1. Go to the Classes Tab.
  2. Highlight the class whose attached constraints you wish to see.
    The attached constraints appear in the Constraints window at the upper right of the Class Form for the class.
