Displaying Attached Queries at a Class

Some queries are attached to a specific class. For this to work properly, the class should be made queriable. If the class is queriable, attached queries are displayed in the Class Form for that class. 

To see the queries attached to a queriable class:

  1. Make sure the class is queriable. See Making a Class Queriable for more information.
  2. Go to the Classes Tab.
  3. Highlight the class whose attached queries you wish to see.
    The attached queries appear in the SLOT-QUERIES window in the Class Form for the class.
