The PAL Queries Tab

The PAL Queries tab allows you to view, create, edit, and evaluate queries. To configure your project to display this tab, see Enabling PAL Queries and Enabling the PAL Constraints and PAL Queries Tabs.

 

 

The PAL Queries Tab has the following areas:

  1. The Choose Queries Pane, which allows you to display and evaluate queries.
  2. The Queries Attachments Pane, which allows you to view the list of classes attached to the selected query.
  3. The Queries Tracing Controls pane, which allows you to trace a built-in function or predicate during evaluation.
  4. The Query Warnings and Errors pane, which displays syntax validation warnings for a selected query.
  5. The Query Response pane for Queries, which displays the list of frames that satisfy a selected query.
