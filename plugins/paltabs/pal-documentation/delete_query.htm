Deleting a Query

You can delete a query from the knowledge base in one of two ways.

  • Directly delete a query at the Queries Tab 
  • Delete the query as an instance of PAL-QUERY at the Instances Tab

At the Queries Tab

At the Queries tab, you may not see the query you wish to delete. In this case, you will have to make sure it is displayed.

  1. Go to the PAL Queries Tab
  2. In the Choose Queries pane, click on the query you wish to delete. 
    OR
  3. If the query you wish to edit is not visible in the Choose Queries pane, click the  Add button to view a list of queries, then choose the one you want and click OK.
  4. Click the Delete button at the upper right of the Choose Queries pane.
    The query is deleted from the knowledge base.

Note: Because selections are not highlighted in the Choose Queries pane, you may wish to delete queries from the Instances tab to ensure you are deleting the query you want to delete. See the next section.

At the Instances Tab

  1. Go to the Instances Tab.
  2. Display the classes under :SYSTEM-CLASS, display the classes under QUERY, and click on PAL-QUERY to highlight it. 
  3. Go to the Direct Instances pane and select the query you wish to delete.
  4. Click the Delete   button at the upper right of the Direct Instances pane.
    The query is deleted from the knowledge base.
